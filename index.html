bsteeleMusicApp Tutorial

This is a tutorial for the bsteeleMusicApp. See bsteeleMusicApp. It's a web app designed to allow musicians to play and sing songs together easily. Think of it as karaoke with chords.

The following is a hierarchical list of operations that you might see when running the app. Tutorial videos on features will be available as links.

Note: It will be a while until we have a complete set of tutorials. So be patient but make suggestions.

Homepage

  • Songlist
    • Scrolling
      Scrolling will roll the song list. Selecting a song will display it on the player screen.
    • Search
      Type search terms in the text field. Searches will be executed immediately. Note that it will include matches from song titles, artists and cover artists.
      Advanced feature: Regular expressions will be processed. For example, try: ba.*fi
    • Filters
      • Filters based on metadata can be used here to limit the selection of songs.
    • Order
      • The order of the song list can be selected.

Concepts

  • Display
    The display and its fonts seem a bit large. The whole presentation is designed to be readable on an HDTV 20 feet away.
  • Leader/Follower
  • Leader Follower Complications
    Often times the leader/follower feature can get in the way. You can leave your tablet ready for the CJ studio, but when you try to use it at home, some features are missing since you are still a "follower." Check the Options host selection if this is the case.
  • Singers Requester

Other complications

  • Use of the app on tablets
    The application is intended to be used on a laptop or desktop. Typically on chrome but many other browsers will work.
    Most all functions of the app will work on most tablets. Complications can arise if:
    1. The width of the tablet is not wide enough to display all the functions. Sometimes rotating the tablet can fix this.
    2. The operating system struggles to allow applications, in this case the browser, to work with files.
  • Use of the app on cell phones
    This can function, but the app struggles to display anything reasonable on such a small screen.
  • Audio
    Unfortunately the audio features of the app are only available on the web (browser) platform.
  • Native versions
    The app is typically used as a web browser application, but there are other formats that the app is available in. See: Options, About, "Download the app".

Beta Features

  • Singers play list bunching
  • Sheet Music Screen
    This screen has actually been around for a long time. Choose a song, edit the song, select "Sheet Music." The top of the screen is meant to be a representation of a bass guitar. Place the mouse over the sheet music on the bottom and try scrolling!
